THIS IS THE USER GUIDE MANUAL FOR HASHER.RB

As of 23/11/25 (English formatting of date) types of hashes available are :

i. General Data
    1. name
    2. surname
    3. age
    4. profession

ii. Student Marksheet
    1. student name
    2. student roll no.
    3. math grade
    4. science grade
    5. social science grade
    6. p.e. grade
    7. percentage
    8. pass / fail

iii. Contact Information
    1. name
    2. phone
    3. email
    4. address
    5. city
    6. zipcode

iv. Employee Record
    1. employee id
    2. name
    3. department
    4. salary
    5. position
    6. joined date

v. Product Inventory
    1. item name
    2. item code
    3. quantity
    4. price
    5. category

vi. Library / Book Entry
    1. title
    2. author
    3. genre
    4. pages
    5. published year

vii. Medical Patient Record
    1. patient name
    2. age
    3. symptoms
    4. diagnosis
    5. medicine
    6. next visit

viii. Game Character Stats
    1. name
    2. level
    3. hp
    4. attack
    5. defense
    6. xp
